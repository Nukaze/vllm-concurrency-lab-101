=============================================================
  Alternative Models for vLLM Concurrency Lab
  (drop-in replacements for Qwen/Qwen2.5-1.5B-Instruct)
=============================================================

HOW TO USE
----------
1. Copy a Model ID below
2. In lab.ipynb, find this line in the Setup cell:

     MODEL_ID = 'Qwen/Qwen2.5-1.5B-Instruct'

   Replace the value with your chosen Model ID, then rerun all cells.

3. To browse the model on HuggingFace, copy the URL and open in browser.

NOTE: Models marked [GATED] require you to:
  - Accept the license on the HuggingFace model page
  - Run: huggingface-cli login   (paste your HF token)
  before running the notebook. Not recommended for quick workshop use.

VRAM REFERENCE (T4 = 15GB, adjust --gpu-memory-utilization if needed)
----------------------------------------------------------------------
  0.5B  model ~ 1-2GB weights  -> very safe, plenty of KV cache room
  1.1B  model ~ 2-3GB weights  -> safe
  1.5B  model ~ 3-4GB weights  -> safe  (default lab setting)
  1.7B  model ~ 3-4GB weights  -> safe
  2B    model ~ 4-5GB weights  -> safe
  3.8B  model ~ 7-8GB weights  -> reduce --gpu-memory-utilization to 0.70
