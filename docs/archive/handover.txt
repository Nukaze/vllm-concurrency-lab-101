WHAT THE LAB DOES
------------------
3 experiments comparing HF Transformers (naive sequential) vs vLLM:
  Exp A - Single request latency (time-to-completion)
  Exp B - Concurrent throughput (N users simultaneously, tokens/sec)
  Exp C - Output accuracy diff (same weights = near-identical outputs)

Model: Qwen/Qwen2.5-1.5B-Instruct (fixed factor, same for both engines)
